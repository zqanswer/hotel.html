我做的网页是关于Telluride度假村的，网页的主要特色是简约欧美风格，网页主要介绍了度假村的特色风景、Telluride的在新闻上的知名度、在度假村能
享受到的活动以及预定页面。在这次制作网页的过程中，我学会了如何构思页面布局，如何完成流畅美观的排版，如何更加熟练地使用轮播图效果，网页间的跳转链接都在
页面顶部，logo以及后面四个链接都一一对应于一个网页，你可以通过点击网页相应的名字到达该网页。
index.html页面使用了轮播图技术，无序列表，以及html的悬停效果HOVER 覆盖。是起首页的功能。
1.html，2.html页面使用了按键hover技术，主要运用排版实现本页面简约干净的效果。
3.html，4.html页面使用了select表单控件以及复选框和submit按键的hover效果。以及为实现预定度假村往返日程的datetime-local类型控件。
